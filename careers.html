Full Stack Developer Intern

We are looking for a Frontend Designer/Developer intern. The ideal candidate is a highly motivated recently graduated student interested in pursuing a career in web development and is excited to work with our in-house development team.

You will be working with our back-end and front-end web developers building custom websites and applications, using HTML 5, CSS, Angular JS, Boot Strap, SAS etc for some of the world’s best known brands. This internship is a full time based opportunity. This is an unpaid internship. We welcome foreign students who can acquire work authorization to apply for all of our internships.

Skills:

  • Knowledge in JavaScript, including DOM manipulation and the JavaScript object model.
  • Thorough understanding of React.js and its core principles.
  • Knowledge with popular React.js workflows (such as Flux or Redux).
  • Familiarity with newer specifications of EcmaScript.
  • Good understanding with data structure libraries (e.g., Immutable.js).
  • Knowledge of isomorphic React is a plus.
  • Familiarity with RESTful APIs.
  • Knowledge of modern authorization mechanisms, such as JSON Web Token.
  • Familiarity with modern front-end build pipelines and tools.
  • Experience with common front-end development tools such as Babel, Webpack, NPM, etc.
